# Big Data Spatial - Rapport Interactif

## Visualisation du Rapport

Pour consulter le rapport complet, ouvrez simplement le fichier **`index.html`** dans votre navigateur web.

## Navigation

Le rapport est organisé en sections thématiques accessibles via le menu de navigation :
- **Introduction** : Contexte et enjeux du Big Data spatial
- **Météorologie** : Prévisions et modèles IA
- **Télécommunications** : Constellations LEO et connectivité
- **Observation de la Terre** : Imagerie et Data Cubes
- **Navigation GNSS** : Positionnement de précision
